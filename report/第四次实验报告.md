# 第四次实验报告：交通标志检测

## 一、实验目的
使用YOLOv8模型进行交通标志目标检测，实现高精度的交通标志识别与定位。

## 二、实验环境
- 硬件：NVIDIA GeForce RTX 4060 Laptop GPU
- 软件：Python 3.12，PyTorch，Ultralytics YOLOv8
- 数据集：交通标志检测数据集（YOLO格式）

## 三、实验方法

### 3.1 数据准备
- 数据集采用标准YOLO格式
- 包含15个类别：
  0. 绿灯
  1. 红灯
  2. 限速10
  3. 限速100
  4. 限速110
  5. 限速120
  6. 限速20
  7. 限速30
  8. 限速40
  9. 限速50
  10. 限速60
  11. 限速70
  12. 限速80
  13. 限速90
  14. 停止标志

### 3.2 模型选择
使用YOLOv8n（nano版本）作为基础模型，该模型具有快速且高效。

### 3.3 训练配置
- 预训练模型：YOLOv8n
- 训练轮数：50 epochs
- 图像尺寸：416x416
- 优化器：自动选择
- 学习率：自适应调整

## 四、实验结果

### 4.1 训练过程
训练在RTX 4060 Laptop GPU上进行，总训练时间约2242秒。

### 4.2 性能指标

| 指标 | 数值 |
|------|------|
| mAP@0.5 | 94.43% |
| mAP@0.5-0.95 | 81.23% |
| Precision | 94.84% |
| Recall | 87.81% |

### 4.3 训练曲线
- 训练损失持续下降，表明模型学习有效
- 验证mAP持续提升，在第50个epoch达到最佳性能
- 没有明显的过拟合现象

## 五、实验分析

### 5.1 模型优势
1. 检测精度高，mAP@0.5达到94.43%
2. 模型收敛快，训练效率高
3. 使用GPU加速，推理速度快

### 5.2 改进方向
1. 可以尝试使用更大的模型（如YOLOv8s/m/l）来提升精度
2. 可以尝试数据增强进一步提升模型泛化能力
3. 可以调整置信度阈值来优化检测效果
